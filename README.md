Honeypot Setup and Monitoring using KFSensor

Overview

This project focuses on setting up and monitoring a honeypot environment using KFSensor. The objective was to understand how honeypots can be used to detect, monitor, and analyze unauthorized access attempts and suspicious network activities.

Tools Used

* KFSensor
* Windows Environment
* Kali Linux VM

Key Activities

* Configured and deployed a honeypot using KFSensor.
* Simulated network services to attract potential attackers.
* Monitored connection attempts and logged suspicious activities.
* Analyzed captured data to understand common attack patterns.

Learning Outcomes

* Understanding of honeypot technology and cybersecurity concepts.
* Exposure to network monitoring and intrusion detection techniques.
* Practical experience in analyzing security-related logs and events.
